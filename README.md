# Tarush Chauhan Portfolio

A 3D developer portfolio for Tarush Chauhan, built with React, TypeScript, Vite, Three.js, React Three Fiber, and GSAP.

This portfolio highlights Tarush's background in backend development, machine learning, cloud technologies, and 3D creative work.

## Highlights

- Animated single-page portfolio inspired by the referenced 3D portfolio design
- 3D scene integration with React Three Fiber and Three.js
- Personalized sections for about, experience, skills, projects, education, certifications, and contact
- Custom SVG project covers and tech badges tailored to Tarush's profile
- Ready for GitHub Pages deployment through GitHub Actions

## Local Development

### Prerequisites

- Node.js 18 or later
- npm 9 or later

### Run locally

```bash
npm install
npm run dev
```

### Production build

```bash
npm run build
```

## Deployment

This repository includes a GitHub Actions workflow for GitHub Pages.

Once the repo is pushed to GitHub:

1. Open repository `Settings`
2. Go to `Pages`
3. Set `Source` to `GitHub Actions`
4. Push to the `main` branch

GitHub Actions will build and deploy the site automatically.
